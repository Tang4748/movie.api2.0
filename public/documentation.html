Documentation Page

INSTRUCTIONS:

This app is designed to help you in the world of best movies ever made!

Here you will find information about different movies, directors, and genres.

You can register, update personal info, and create a list of favorite movies.

ENDPOINT DOCUMENTATION:

Request URL HTTP Method Response
Return a list of ALL movies to the user /movies GET a JSON object holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/title GET a JSON object holding data about a single movie containing the data above
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/title/genreName GET JSON object holding data about genre
Return data about a director (bio, birth year, death year) by name /movies/title/directorName GET JSON object holding data about director
Allow new users to register /users POST a text message indicating successfull registration of the account
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/id/favoriteMovieTitle POST a text message that movie was added to favourites list
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/id/favoriteMovieTitle DELETE text message that movie was removed from favorites list
Allow existing users to deregister (showing only a text that a user had been removed) /users/id DELETE a text message indicating successfull deletion of the user account
